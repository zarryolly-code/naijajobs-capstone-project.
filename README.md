# NaijaJobs Capstone Project

## Overview

NaijaJobs is a responsive job listing web application designed to connect job seekers with employers across Nigeria.

The platform allows users to browse job opportunities, search for jobs, view detailed job information, and post new job listings through an easy-to-use interface.

This project was developed using HTML, CSS, and JavaScript as part of a Software Development Capstone Project.

---

## Features

- Browse available job listings
- Search jobs by title or company
- View detailed job information
- Post new job opportunities
- Responsive user interface
- Local Storage support integration
